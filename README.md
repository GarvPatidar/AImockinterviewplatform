# AI Mock Interview Platform

An AI-powered mock interview platform that helps users prepare for technical interviews with real-time AI-generated questions, performance tracking, and interview feedback.

## 🚀 Features

* Google Authentication using Firebase
* AI-powered mock interviews
* Real-time interview experience
* User profile management
* Performance tracking and analytics
* Secure backend authentication
* Responsive and modern UI

## 🛠️ Tech Stack

### Frontend

* React.js
* Redux Toolkit
* Tailwind CSS
* Axios
* Firebase Authentication

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

### AI Integration

* Google Gemini API / AI Model

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/GarvPatidar/AImockinterviewplatform.git
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### Environment Variables

Create `.env` files in frontend and backend.

Frontend:

```env
VITE_FIREBASE_APIKEY=YOUR_FIREBASE_API_KEY
```

Backend:

```env
MONGODB_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
GEMINI_API_KEY=YOUR_GEMINI_KEY
```

### Run Application

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```
